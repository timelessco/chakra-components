# Text

The `Text` component allows you to include texts everywhere expect the headings. This is wrapper around HTML `p` tag.

HTML DOM structure

```
<p>String</p>
```

## Import

```
import Text from 'chakra-wrapper';
```

## Props

- `override` - Object to be passed in-order to override the base styles of the checkbox.

### Allowed style props in `override`

- Color
- Space
- Typography

## Props in action

### `override`

<Playground>
  <Text override={{ color: 'whatsapp.500' }}>Color Override</Text>
</Playground>
