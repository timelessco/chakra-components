# Button

Button component wraps the Chakra Button component and by default rendered as button.

The HTML DOM structure is

```
<Button>Text</Button>
```

## Usage

<Playground>
  <Button variantColor="green">Button</Button>
</Playground>

---

## Button Sizes

Use the size prop to change the size of the button. You can set the value to xs, sm, md, or lg

<Playground>
  <Flex.Row alignItems="center" justifyContent="space-evenly">
    <Button variantColor="teal" size="xs">
      Button
    </Button>
    <Button variantColor="teal" size="sm">
      Button
    </Button>
    <Button variantColor="teal" size="md">
      Button
    </Button>
    <Button variantColor="teal" size="lg">
      Button
    </Button>
  </Flex.Row>
</Playground>

---

## Button Variant

Use dot notation to create solid, ghost, outline, or link buttons.

<Playground>
  <Flex.Row alignItems="center" justifyContent="space-evenly">
    <Button.Solid>Button</Button.Solid>
    <Button.Outline>Button</Button.Outline>
    <Button.Ghost>Button</Button.Ghost>
    <Button.Link href="#button-variant">Button</Button.Link>
    <Button.Link href="https://chakra-ui.com/" isExternal>
      Button
    </Button.Link>
  </Flex.Row>
</Playground>

---

## Button with Icon

Use dot notation to create left & right icon buttons.

<Playground>
  <Flex.Row alignItems="center" justifyContent="space-evenly">
    <Button.LeftIcon>Call us</Button.LeftIcon>
    <Button.RightIcon>Email</Button.RightIcon>
  </Flex.Row>
</Playground>

---

## More Button Variant

Use dot notation to create your own set of Primary and Secondary buttons apart from the variants.

<Playground>
  <Flex.Row alignItems="center" justifyContent="space-evenly">
    <Button.Primary isLoading>Primary Button</Button.Primary>
    <Button.Secondary>Secondary Button</Button.Secondary>
  </Flex.Row>
</Playground>

---

## Button loading state

Pass isLoading prop to the Button component to show it's loading state. You can optionally pass loadingText prop, if you do, the button will show a spinner and the loading text. Otherwise, the button will take the width of the text label and show only the spinner

<Playground>
  <Flex.Row alignItems="center">
    <Button
      isLoading
      variantColor="teal"
      variant="solid"
      leftLoadingIcon
      loadingIconSpace={2}
    >
      Email
    </Button>
    <Button
      ml={2}
      isLoading
      loadingText="Submitting"
      variantColor="teal"
      variant="outline"
      rightLoadingIcon
      loadingIconSpace={2}
    >
      Submit
    </Button>
  </Flex.Row>
</Playground>
