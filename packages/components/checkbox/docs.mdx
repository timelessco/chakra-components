# Checkbox

Checkbox component that wraps Chakra

The HTML DOM structure is

```
<label>          // Wraps the entire checkbox (override - wrapper)
  <input />      // Hidden input to carry the value / states like focused
  <div />        // contains a div with <svg /> (override - controlBox)
  <span />       // Holds the label (override - label)
</label>
```
