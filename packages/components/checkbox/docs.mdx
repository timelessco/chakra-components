# Checkbox

The `Checkbox` component is a HTML `input` with type checkbox built using Chakra Components.

HTML DOM structure

```
<label>          // Wraps the entire checkbox
  <input />      // Hidden input to carry the value / states like focused
  <div />        // contains a div with <svg />
  <span />       // Holds the label
</label>
```

## Import

```
import Checkbox from 'chakra-wrapper';
```

## `<Checkbox>`

<Playground>
  <Checkbox>Default Checkbox</Checkbox>
</Playground>

## Props

- `id` - The id assigned to input field
- `name` - The name of the input field in a checkbox (Useful for form submission)
- `value` - The value to be used in the checkbox input. This is the value that will be returned on form submission.
- `isDisabled` - If true, the checkbox will be disabled.
- `isReversed` - If true, the checkbox will be reversed.
- `defaultIsChecked` - If true, the checkbox will be initially checked.
- `isChecked` - If true, the checkbox will be checked. You'll need to pass onChange to update it's value, since it's controlled
- `onChange` - Function called when the state of the checkbox changes.
- `onBlur` - Function called when you blur out of the checkbox.
- `onFocus` - Function called when the checkbox receive focus.
- `override` - Object to be passed in-order to override the base styles of the checkbox.

### Allowed style props in `override`

- `wrapper` - layout, typography & flex
- `controlBox` - layout, border, borderRadius, color, shadow, \_hover, \_focus, \_disabled, \_checked & \_checkedAndDisabled
- `label` - color, space & typography
