# Box

Box component wraps the Chakra PseudoBox component and by default rendered as div, but can be change by passing 'as' prop 
(as: 'input')

The HTML DOM structure is

```
  <div></div> 
```

## Usage

<Playground>
  <Box bg="tomato" w="100%" p={4} color="white">
    This is the Box
  </Box>
</Playground>

___
### As input and button
<Playground>
  <Flex as="form" p="1">
    <Box
      as="input"
      placeholder="Your email"
      type="email"
      flex="1"
      py={2}
      px={4}
      rounded="md"
      bg="gray.100"
      borderWidth="1px"
      _hover={{ borderColor: "gray.200", bg: "gray.200" }}
      _focus={{
        outline: "none",
        bg: "white",
        boxShadow: "outline",
        borderColor: "gray.300",
      }}
    />
    <Box
      as="button"
      bg="teal.500"
      py={2}
      px={4}
      ml={3}
      rounded="md"
      fontWeight="semibold"
      color="white"
      _hover={{ bg: "teal.600" }}
      _focus={{ boxShadow: "outline" }}
    >
      Sign Up
    </Box>
  </Flex>
</Playground>

## Hover

Add the _hover prop to only apply style props on hover.

<Playground>
  <Box
    as="button"
    color="blue.700"
    fontWeight="semibold"
    py={2}
    px={4}
    borderWidth="1px"
    borderColor="blue.500"
    rounded="md"
    _hover={{ bg: "blue.500", color: " white" }}
    _focus={{ boxShadow: "outline" }}
  >
    Hover me
  </Box>
</Playground>
