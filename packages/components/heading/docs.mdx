# Heading

The `Heading` component allows you to include headings h1 & h2. Default `Heading` is rendered as HTML `h1` tag.

HTML DOM structure

```
<h1>String</h1>
```

## Import

```
import Heading from 'chakra-wrapper';
```

## `<Heading>`

<Playground>
  <Heading>Default Heading without any styles</Heading>
</Playground>

## `<Heading.H1>`

<Playground>
  <Heading.H1>Styled H1 Heading</Heading.H1>
</Playground>

## `<Heading.H2>`

<Playground>
  <Heading.H2>Styled H2 Heading</Heading.H2>
</Playground>

## Props

- `override` - Object to be passed in-order to override the base styles of the checkbox.

### Allowed style props in `override`

- Color
- Space
- Typography

## Props in action

### `override`

<Playground>
  <Heading.H1 override={{ fontSize: '6xl' }}>Font Size Override</Heading.H1>
</Playground>
