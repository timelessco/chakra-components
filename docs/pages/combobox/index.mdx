# ComboBox

An accessible ComboBox with single, multi & accessible select with typeahead &
autocomplete support.

## Import

```js
import { ComboBox } from "@chakra-components/core";
```

## Usage

Here's a basic usage of the Select component.

```jsx
<ComboBox>
  <ComboBoxLeftElement color="gray.300" fontSize="1.2em" children="$" />
  <ComboBoxRightElement
    children={<Icon name="chevron-down" fontSize="1.5rem" />}
  />
  <ComboBoxClearElement />
  <ComboBoxInput as={Select} placeholder="Enter amount">
    <option value="option1">Option 1</option>
    <option value="option2">Option 2</option>
    <option value="option3">Option 3</option>
  </ComboBoxInput>
</ComboBox>
```
