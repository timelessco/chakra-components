# Combobox



## Import



```js
import {
  MySelect
} from "@chakra-components/core";
const options = [
  "Acai",
  "Apples",
  "Apricots",
  "Avocado"
];
```

## Basic Usage

```jsx


<MySelect
  value={"Tomatoes"}
  options={[
  "Acai",
  "Apples",
  "Apricots",
  "Avocado"
]}
  onChange={() => {}}
  placeholder="Select one..."
/>

```
