# Checkbox

Checkbox component that wraps Chakra

The HTML DOM structure is

```
<label>          // Wraps the entire checkbox (override - wrapper)
  <input />      // Hidden input to carry the value / states like focused
  <div />        // contains a div with <svg /> (override - controlBox)
  <span />       // Holds the label (override - label)
</label>
```

## Basic usage
---

#### Default Checkbox
<Playground>
  <Checkbox
  >Click me</Checkbox>
</Playground>

---
---

#### Disabled Checkbox
<Playground>
  <Checkbox
    isDisabled={true}
  >Click me</Checkbox>
</Playground>

---
---

#### Reversed Checkbox
<Playground>
  <Checkbox
    isReversed={true}
  >Click me</Checkbox>
</Playground>

---
---

#### Checkbox with custom color
<Playground>
  <Checkbox>This is the normal box</Checkbox>
  <Checkbox override={{
    wrapper: { lineHeight: 5 },
    controlBox: { w: 10, h: 10 }
  }}>
    This is a bigger box with more line height
  </Checkbox>

  <Checkbox override={{ controlBox: { w: 20, h: 20 }}}>This is a much bigger box</Checkbox>

  <Checkbox override={{
    controlBox: { w: 20, h: 20 },
    label: { fontSize: 40 }
  }}>This is a much bigger box with bigger text</Checkbox>
</Playground>
